  1. Parameter estimation : comparing the LS method with the result from lm()

Lets start by creating the necessary matrices

# y = response variable
y <- bread$height
# X = design matrix
X <- matrix(nrow = 12,ncol = 3)
X[,1] <- rep(1,12)
X[,2] <- c(rep(0,4),rep(1,4),rep(0,4))
X[,3] <- c(rep(0,8),rep(1,4))

XtX <- t(X)%*%X

XtX.inv <- solve(XtX)

beta.hat <- XtX.inv%*%t(X)%*%y

  1. Alternatively, we can estimate the contrasts between effects \(\tau_i\)
t1.t2<-fit.contrast(model = mod0, 'time', c(1,-1,0))
t1.t3<-fit.contrast(model = mod0, 'time', c(1,0,-1))
  1. Hypothesis testing , we now want to test the hypothesis \(H_0 : \mu_1 = \mu_2 = \cdots = \mu_t\) vs the alternative, which is unequal effect for any pair. F test looks at the ratio of variation from two different sources. In our case, we are interested in treatment variation vs model variation. In R, we can use lm() to do ANOVA, but aov is a simple way that is sometimes preferred.
mod1 <- aov(height~time, data = bread)
print(summary(mod1))
##             Df Sum Sq Mean Sq F value Pr(>F)

In the F-test above, assuming a significance level \(\alpha = 0.5\), we should reject \(H_0.\)

  1. Model assumption check - how valid are the results of our analysis so far? One way to find out is the verify that the model assumptions are met, which we can show in R using the following commands:
# generate 2 by 2 canvas for plots
